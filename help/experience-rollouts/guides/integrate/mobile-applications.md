# Mobiele toepassingen {#mobile-applications}

De mobiele toepassingen integreren met de Rollouts van de Ervaring door de **Eigenschap API V3** te roepen om eigenschapvlaggen terug te winnen en voorwaardelijke logica in uw toepassingscode toe te passen.

Zie **de Eigenschap API van GET V3** in de Eigenschap API sectie van deze gids voor de volledige API verwijzing, met inbegrip van verzoekparameters, reactieformaat, en voorbeeldvraag.
